# 🌾 Ekosistem Digital Indramayu Club (hugoNUR3)
Selamat datang di gerbang repositori resmi digital **Indramayu Club** yang dikelola langsung oleh Komandan **Jamhari-Dulkohar**. Repositori ini berjalan di atas sistem otomatisasi **NUR_ENGINE_GATEWAY** untuk melayani media pembelajaran mandiri, arisan digital, dan interaksi lisan.

---

## 🏛️ Struktur Navigasi Gerbang Ruangan NUR
Ekosistem ini terdiri dari sepuluh kamar utama yang terintegrasi penuh:
* **Nur 1:** `nur1_media.html` — INDRAMAYU_CLUB (Akademi Android)
* **Nur 2:** `nur2_login.html` — LOGIN SULTAN / SERVER RAJA / ADMIN VIDEO
* **Nur 3:** `nur3_keuangan_pagi.html` — KEUANGAN PAGI / KAS UTAMA (Potongan 5%)
* **Nur 4:** `nur4_aset_digital.html` — KREATOR ASET DIGITAL
* **Nur 5:** `nur5_menu_utama.html` — GAME ARISAN & UNDIAN SULTAN
* **Nur 6:** `03_ADM_KEUANGAN/` — VAULT BANK GAME (Penyimpanan Koin/Voucher)
* **Nur 7:** `nur7a.streaming.html` & `nur7b_live_member.html` — STREAMING A & LIVE MEMBER B
* **Nur 8:** `nur8_emulator.html` — EMULATOR & KUMPULAN GAME JADUL
* **Nur 9:** `nur9_index2222.html` — PROTOKOL SINKRONISASI & BECKUP DATA
* **Nur 10:** `nur10_kamus_alwi.html` — PORTAL KAMUS ALWI VVIP (Wangsit Basa Lisan)
