*   **Pukul 08:00 - 12:00 (Sesi Pagi):** **NUR 2 & NUR 3** bertugas penuh dalam mengaudit **Kas Otomatis 5%** dan melakukan validasi *Voucher* harian untuk menjaga stabilitas sirkulasi keuangan.

*   **Di Luar Jam Operasional:** **NUR 7** bersiap siaga (*standby*) di Hub Internasional untuk menyapa anggota (*member*) baru yang bergabung dengan kalimat puitis dan memantau lalu lintas jaringan global.
